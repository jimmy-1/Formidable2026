# Formidable 2026

Formidable 2026 是一款基于 C++/C/Python 开发的高性能、模块化远程管理平台。

## 核心特性

- **双架构兼容性**: 支持 Win32 和 x64 平台混合控制，x64 主控可无缝管理 Win32 被控端，反之亦然。
- **模块化架构**: 采用 `Master (GUI) + Client (Loader) + Modules (DLLs)` 模式。核心功能均通过 DLL 插件形式动态加载，提升了系统的灵活性和可维护性。
- **内存加载技术**: 客户端支持通过内存加载 DLL 模块，减少磁盘痕迹，增强隐蔽性。
- **自定义通信协议**: 基于 TCP/Socket 封装的自定义私有协议，包含心跳保活、RTT 计算及实时状态更新。
- **无第三方依赖**: 核心代码尽量避免引入不可控的第三方二进制文件，所有外部引用均确保来源透明可控。

## 功能概览

| 模块 | 功能描述 | 状态 |
| :--- | :--- | :--- |
| **终端管理** | 远程 CMD/PowerShell 执行，支持管道实时回显及交互。 | 100% |
| **进程管理** | 进程/模块遍历、结束进程、路径复制，支持跨架构兼容。 | 100% |
| **窗口管理** | 顶层窗口枚举、隐藏/显示、状态控制（最大化/最小化/关闭）。 | 100% |
| **桌面监控** | 实时屏幕截图、监控及远程鼠标键盘控制。 | 100% |
| **文件管理** | 磁盘枚举、文件上传/下载/删除/重命名、远程管理员运行。 | 100% |
| **生成服务** | 集成配置编辑器，支持将配置信息直接注入生成的 Stub。 | 100% |
| **服务管理** | 系统服务列表枚举、控制（启动/停止/删除）。 | 50% |
| **注册表管理** | 注册表项树状浏览、键值对编辑。 | 50% |
| **系统信息** | 获取主机硬件、系统版本、网络状态及地理位置查询。 | 80% |
| **多媒体管理** | 音频监听及摄像头监控框架。 | 30% |

## 项目结构

```text
.
├── Master/             # 主控端 GUI 源代码 (Visual Studio 工程)
├── Client/             # 被控端 Loader 源代码
├── Modules/            # 功能模块 DLL 源代码
│   ├── Terminal/       # 终端管理模块
│   ├── FileManager/    # 文件管理模块
│   └── ...             # 其他模块
├── Common/             # 公用头文件、配置及工具类
└── Formidable2026.sln  # Visual Studio 解决方案文件
```

## 开发环境

- **操作系统**: Windows 10/11
- **开发工具**: Visual Studio 2022 (或更高版本)
- **编程语言**: C++17, C, Python (可选集成)
- **字符编码**: 统一采用 `UTF-8 with BOM` 格式，确保跨环境显示正常。

## 编译指南

1. 使用 Visual Studio 打开 `Formidable2026.sln`。
2. 确保已安装 C++ 桌面开发组件。
3. 选择对应的配置（Debug/Release）和平台（x86/x64）。
4. 先编译 `Modules` 目录下的所有 DLL 工程，然后编译 `Client`，最后编译 `Master`。
5. 生成的可执行文件位于各工程的 `Intermediate` 目录下或解决方案根目录的输出路径。
